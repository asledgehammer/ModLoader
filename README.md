# ModLoader

- Merge the contents of the `Zomboid` folder in the ``../Zomboid/`` folder on the server.
``../Zomboid/Lua/mod_loader/mods/..``

- To apply mods using ModLoader, place their files under the folder:
`Zomboid/Lua/Mmod_loader/mods/{the_mod_id}/..`

Check the ModLoader-Example mod for an example folder to apply for mods.
